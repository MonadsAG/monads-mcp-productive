Aktualisiere die Productive.io API-Spec:

1. Führe den Scraper aus: `cd docs/api-spec && python productive_to_openapi.py`
2. Prüfe den Output (Anzahl Resources, Paths, Operations)
3. Lies den neuesten Eintrag in `docs/api-spec/CHANGELOG.md` und fasse die Änderungen zusammen
4. Falls es Änderungen gab, frage ob committed werden soll
